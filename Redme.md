1️.What is the difference between var, let, and const? 
Ans:
var= Function scoped. It can be redeclared and its value can be changed.
let= Block scoped. It cannot be redeclared but its value can change.
const = Block scoped. Its value cannot be changed after it is assigned.

2 . What is the spread operator (...)?
Ans:
The spread operator (...) is used to copy or expand values from an array or object into another array or object.

3 . What is the difference between map(), filter(), and forEach()?
Ans:
map() → Creates a new array by changing each element.
filter() → Creates a new array with elements that match a condition.
forEach() → Runs a function for each element but does not create a new array.

4 . What is an arrow function?
Ans:
An arrow function is a short way to write a function in JavaScript using the => symbol.

5 . What are template literals?
Ans:
Template literals are strings written with backticks (`). They allow adding variables inside a string easily.
